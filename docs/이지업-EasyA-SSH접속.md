# 이지업 서버호스팅 Easy A – SSH 접속 가이드

## 1. 카페24에서 할 일

### 1) SSH(Shell) 접속 허용
- **나의 서비스관리** → **호스팅관리** → **FTP/Shell 접속설정** (또는 서버 상세 메뉴)
- **Shell(SSH) 접속** 을 **허용** 으로 변경
- 접속 허용 범위: **국내만** / **특정 IP** / **전체** 중 선택 (테스트 시엔 전체 허용 후 나중에 제한 권장)

### 2) 접속용 IP 확인
- **172.235.192.86** 은 사설(내부) IP일 수 있음
- 같은 호스팅 관리 화면에서 **공인 IP**, **외부 IP**, **접속 IP** 항목 확인
- 표시된 **공인 IP**로 SSH 접속 (예: 121.x.x.x, 211.x.x.x 등)

### 3) 접속 계정/비밀번호
- **FTP 접속 ID·비밀번호** 가 SSH 로그인과 동일한 경우가 많음
- 이지업 서버호스팅 전용 **관리자 계정** 이 있다면 해당 계정 사용 (콘솔 안내 확인)

---

## 2. 접속 명령 (Windows PowerShell)

```powershell
# 공인 IP 확인 후 (ubuntu가 아니면 FTP ID 또는 안내된 계정명 사용)
ssh 계정명@공인IP
# 예: ssh myid@121.165.xx.xx
```

비밀번호는 FTP 비밀번호 또는 서버 관리자 비밀번호 입력.

---

## 3. 그래도 접속이 안 될 때

| 확인 항목 | 설명 |
|----------|------|
| SSH 허용 | FTP/Shell 접속설정에서 Shell(SSH) **허용** 인지 확인 |
| 공인 IP | 접속에 **공인(외부) IP** 사용했는지 확인 (172.x 는 내부 IP일 수 있음) |
| IP 제한 | “특정 IP만 허용”이면 현재 PC 공인 IP가 포함되는지 확인 |
| 방화벽 | 서버/호스팅 방화벽에서 **22번 포트** 허용 여부 확인 |
| 웹터미널 | 이지업 콘솔에 **웹터미널**/원격접속이 있으면, 그걸로 먼저 접속해 보기 |
